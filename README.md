# triangle_classification

This repository contains a small triangle classification service.

- The service implementation can be found in `TriangleClassification/`, with the classification code in `TriangleClassificationService.cs` and the result type definitions in `TriangleClassificationResult.cs`.
- Tests are located in `TriangleClassification.Tests/TriangleClassificationTests.cs`.
- An example usage is available in the CLI project at `TriangleClassification.Cli/Program.cs`. This runs the service a couple times and prints the results.
